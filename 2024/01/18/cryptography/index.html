one public key and one private key

private key

Encrypted communication: private key use to decrypt the message

digital signature: private key use to encrypt the hash value of message

public key

Generate digital signatures 生成数字签名

make sure message Authenticity and integrity

使用 asymmetric key algorithm 实现

  1. create hash value

    Use hash function SHA-256 to handle message

    genrate a fixed size hash value

  2. Encrypt hash value

    Sender use private key to encrypt the hash value

    the encry value will be the digital signature

  3. add signature

    sender attach the digital signature after the message (一般来说是发送 保密信息)

    or

    sender send to the recevier separately (发送软件 or 系统)

Verify digital signature 验证数字签名

  1. get signature and data
  2. Decrypt signature
  3. Recalculate hash value
  4. compare hash value

将recalcuate hash value 和 decrypt hash value比较

两个hash value一致，说明消息没有被tampered修改，而且是由相应的Private key holder发送的
